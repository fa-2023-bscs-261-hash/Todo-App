# To-Do App

A sleek, responsive To-Do app to organize tasks efficiently. Built with **HTML**, **CSS**, and **JavaScript**, it features a clean interface, smooth interactions, and visual priority tracking.

## Features

* Add, edit, and delete tasks
* Mark tasks as completed
* Priority highlighting
* Responsive design for desktop and mobile

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/todo-app.git
```

2. Open `index.html` in your browser.

## Usage

* Add tasks in the input field.
* Click tasks to mark as completed.
* Delete tasks with the delete button.

## Technologies

* HTML
* CSS
* JavaScript
